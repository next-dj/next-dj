Configuration package
=====================

Per-submodule reference for :mod:`next.conf`. The package reads
``settings.NEXT_FRAMEWORK``, caches the merged view, and emits a
``settings_reloaded`` signal when the cache is invalidated. See
:doc:`/content/api/reference` for the ``NEXT_FRAMEWORK`` schema and
:doc:`/content/guide/extending` for a walkthrough of the extension
points.
