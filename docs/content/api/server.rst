Server package
==============

Per-submodule reference for :mod:`next.server`. The package replaces
the Django ``StatReloader`` with a variant that tracks the framework
roots and exposes hooks to contribute extra watch specs. See
:doc:`/content/guide/autoreload` for a narrative walkthrough.
